# 🛍️ E-Commerce Frontend

A modern and responsive e-commerce frontend built with **React** and **Vite**. This project focuses on delivering a clean shopping experience through reusable components, efficient state management, and seamless API integration.

Whether browsing products, managing a cart, or updating inventory, the application is designed with scalability and maintainability in mind.

---

## ✨ Features

- 🏠 Responsive Home page
- 🛍️ Product listing and details
- 🛒 Shopping cart management
- ➕ Add new products
- ✏️ Update existing products
- 💳 Checkout popup/modal
- 🔄 API integration using Axios
- 📱 Responsive UI
- ⚡ Fast performance with React + Vite

---

## 🛠️ Tech Stack

- React
- Vite
- JavaScript (ES6+)
- CSS3
- Axios

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/ecommerce-frontend.git
```

### Navigate to the project

```bash
cd ecommerce-frontend
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```

## 📌 Project Overview

### Home

Displays the available products and serves as the landing page of the application.

### Product

Shows product information including price and available actions.

### Cart

Allows users to review selected products before checkout.

### Checkout Popup

Provides a simple checkout interface through a modal.

### Add Product

Interface for adding new products.

### Update Product

Allows editing existing product information.

### Navbar

Provides navigation across the application.

### Axios Configuration

Handles API requests through a centralized Axios instance.
